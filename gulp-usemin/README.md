# test-gulp-usemin

Example for gulp-usemin.

* Concatenate and minify for CSS/JS
* replace the CSS/JS references from HTML

# Installation

1. Install node.js, gulp ( global ) and bower
1. git clone https://github.com/akabekobeko/examples-web-app.git
1. cd gulp-usemin
1. npm install
1. bower install
1. gulp build
