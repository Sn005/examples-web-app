# test-grunt-usemin

Example for grunt usemin.

* Concatenate and minify for CSS/JS
* replace the CSS/JS references from HTML

# Installation

1. Install node.js, grunt-cli and bower
1. git clone https://github.com/akabekobeko/examples-web-app.git
1. cd grunt-usemin
1. npm install
1. bower install
1. grunt build
