# Example: ES6 unit test

Sample for unit test in ES6.

## Installation & Build

* Install [Node.js](https://nodejs.org/ "Node.js")
* ```git clone https://github.com/akabekobeko/examples-web-app.git```
* ```cd es6-unit-test```
* ```npm i```
* ```npm test```
